# Create-SNOWVulnIncident
author: Nathan Swift

Upon 'Machines should have vulnerability findings resolved' recommendation trigger by a virtual machine this logic app will go out and retrieve the vulnerability findings on the VM and create a ServiceNow Incident Request per each vulnerability finding.
